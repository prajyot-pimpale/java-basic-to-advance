# Java Basic to Advance

A complete Java learning repository covering basics to advanced concepts including OOP, collections, multithreading, exception handling, JDBC, file handling, Java 8+ features, and real-world projects. Designed for beginners and intermediate developers with practical examples and structured learning.

---

## 🚀 Topics Covered

* Java Basics
* Variables & Data Types
* Operators
* Conditional Statements
* Loops
* Arrays & Strings
* Functions & Methods
* Object-Oriented Programming (OOP)
* Inheritance
* Polymorphism
* Abstraction
* Encapsulation
* Exception Handling
* Collections Framework
* File Handling
* Multithreading
* JDBC
* Java 8 Features
* Stream API
* Lambda Expressions
* Mini Projects
* Practice Programs

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/java-basic-to-advance.git
```

2. Open the project in your IDE

3. Compile and run Java files

```bash
javac FileName.java
java FileName
```
